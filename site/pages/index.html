example.ts 
import { GPUContext, BitonicSorter } from 'webgpu-sorting';

// Initialize WebGPU context
const gpu = new GPUContext();
await gpu.initialize();

// Create sorter
const sorter = new BitonicSorter(gpu);

// Sort data on GPU
const data = new Uint32Array([5, 2, 8, 1, 9]);
const { sortedData } = await sorter.sort(data);

// 10-100x faster than Array.sort()!

Sorting Algorithms

Bitonic Sort

O(n log²n)

A comparison-based parallel sorting network. Excellent for general-purpose sorting with predictable performance characteristics.

  • Works with any comparable data type
  • Stable sorting network
  • Best for: General-purpose sorting

Radix Sort

O(n × k)

A non-comparison integer sorting algorithm using parallel histogram and prefix sum operations. Extremely fast for 32-bit integers.

  • Specialized for integer sorting
  • Parallel histogram computation
  • Best for: Large integer arrays
