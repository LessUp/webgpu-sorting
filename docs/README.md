## Documentation principles

- Keep public docs aligned with the actual repository structure.
- Prefer rewriting or consolidating stale docs instead of layering on duplicates.
- Default to English for shared docs; add `.zh.md` translations when they will be maintained.
- Treat GitHub Pages, README, and this docs hub as one connected public narrative.
