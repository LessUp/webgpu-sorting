## Summary

This release focuses on code quality improvements, CI/CD optimization, and comprehensive documentation updates. No breaking changes were introduced.

## Highlights

- Code quality improvements with explicit null checks
- Optimized GitHub Actions workflows
- Enhanced documentation with bilingual support
- Removed unused code files

## Changes

### Fixed

- Removed non-null assertions (`!`) in `BitonicSorter.ts` and `RadixSorter.ts`, replaced with explicit null checks
- Fixed CI workflow branch configuration (`main` → `master`)
- Fixed GitHub Pages workflow branch configuration (`main` → `master`)

### Optimized

- Separated lint, test, and build jobs in CI workflow for better efficiency
- Added path filtering to Pages workflow to trigger builds only on relevant file changes
- Updated ESLint configuration to allow console statements in examples directory

### Removed

- Deleted unused `scan.wgsl` shader file

### CI/CD Enhancements

- Added `.nojekyll` file for proper GitHub Pages deployment handling
- Added build artifact uploads (retained for 7 days)
- Added coverage report uploads (retained for 30 days)
- Optimized job dependencies: lint → test → build

### Documentation

- Fixed `BenchmarkResult` type definition in `docs/API.md`
- Updated project structure in `README.md`, removed `scan.wgsl` references
- Updated `PROJECT_OVERVIEW.md`, removed `scan.wgsl` references
- Updated architecture diagrams in `docs/TECHNICAL.md`
- Updated `.kiro/specs/` specifications

## Migration Guide

No migration required. This is a maintenance release with no API changes.
