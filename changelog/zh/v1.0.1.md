## 概要

此版本专注于代码质量改进、CI/CD 优化和全面的文档更新。未引入破坏性变更。

## 亮点

- 代码质量改进，使用显式空值检查
- 优化 GitHub Actions 工作流
- 增强文档，提供双语支持
- 移除未使用的代码文件

## 变更

### 修复

- 移除 `BitonicSorter.ts` 和 `RadixSorter.ts` 中的非空断言 (`!`)，替换为显式空值检查
- 修复 CI 工作流分支配置 (`main` → `master`)
- 修复 GitHub Pages 工作流分支配置 (`main` → `master`)

### 优化

- 分离 CI 工作流中的 lint、test 和 build 作业以提高效率
- 为 Pages 工作流添加路径过滤，仅在相关文件变更时触发构建
- 更新 ESLint 配置允许 examples 目录使用 console 语句

### 移除

- 删除未使用的 `scan.wgsl` 着色器文件

### CI/CD 增强

- 添加 `.nojekyll` 文件以正确处理 GitHub Pages 部署
- 添加构建产物上传功能（保留 7 天）
- 添加覆盖率报告上传功能（保留 30 天）
- 优化作业依赖关系：lint → test → build

### 文档

- 修复 `docs/API.md` 中的 `BenchmarkResult` 类型定义
- 更新 `README.md` 项目结构，移除 `scan.wgsl` 引用
- 更新 `PROJECT_OVERVIEW.md`，移除 `scan.wgsl` 引用
- 更新 `docs/TECHNICAL.md` 中的架构图
- 更新 `.kiro/specs/` 规格文档

## 迁移指南

无需迁移。这是一个维护版本，没有 API 变更。
